## 🛠️ Installation et développement

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Le site sera accessible sur `http://localhost:5173`

## 🌿 Workflow Git

### Branches principales
- `main` - Version production
- `develop` - Version de développement

### Convention de branches
- `feature/nom-fonctionnalite` - Nouvelles fonctionnalités
- `fix/description-bug` - Corrections de bugs
- `refactor/description` - Refactoring
- `style/description` - Modifications CSS/UI
- `docs/description` - Documentation

### Convention de commits
Format : `type(scope): description`

**Exemples :**
- `feat(about): add professional journey section`
- `feat(projects): create project card component`
- `style(navbar): improve mobile responsiveness`
- `fix(footer): correct social links alignment`
