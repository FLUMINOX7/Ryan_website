# Ryan - Portfolio Professionnel

Portfolio professionnel et interactif présentant mon parcours, mes compétences et mes projets informatiques.

## 🌟 Fonctionnalités

- Design moderne et professionnel
- Interface responsive (mobile, tablette, desktop)
- Navigation fluide avec animations
- Sections complètes : Accueil, À Propos, Compétences, Projets, Contact
- Optimisé pour les performances
- Code propre et maintenable

## 🚀 Utilisation

### Ouvrir le site localement

Simplement ouvrir le fichier `index.html` dans votre navigateur web préféré.

### Personnalisation

1. **Contenu personnel** : Modifier le fichier `index.html` pour ajouter vos informations personnelles
2. **Projets** : Mettre à jour la section "Projets" avec vos propres projets
3. **Compétences** : Adapter la section "Compétences" selon votre expertise
4. **Contact** : Ajouter vos vraies coordonnées de contact (email, LinkedIn, GitHub)
5. **Style** : Personnaliser les couleurs dans `styles.css` (variables CSS dans `:root`)

## 📁 Structure du projet

```
Ryan_website/
├── index.html      # Page principale HTML
├── styles.css      # Feuille de styles CSS
├── script.js       # Scripts JavaScript
└── README.md       # Documentation
```

## 🎨 Technologies utilisées

- HTML5
- CSS3 (avec variables CSS et animations)
- JavaScript vanilla (pas de dépendances)
- Design responsive avec CSS Grid et Flexbox
